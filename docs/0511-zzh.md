```txt
2 P_target = exp(lnP_target) 
```

# 模型B：固定⾼度能耗模型的⾼度扩展

已有固定⾼度能耗模型可以保留，只是把输⼊从：

代码块

路线距离、速度、载重、电池参数、固定⾼度天⽓1

扩展为：

代码块

路线距离、速度、载重、电池参数、⻜⾏⾼度处天⽓1

具体加⼊这些特征：


代码块


```txt
headwind = 逆风分量  
crosswind = 侧风分量  
rho_h = 空气密度  
T_h = 温度  
P_h = 气压  
RH_h = 湿度  
climb_rate = 爬升率  
height_agl = 飞行高度
```

NASA的⼤⽓模型资料也说明，⽓压会随⾼度升⾼⽽下降，空⽓密度由温度和⽓压共同决定，空⽓动⼒直接依赖空⽓密度。 对多旋翼来说，悬停诱导功率与空⽓密度有关，理想诱导功率可写成与 1 /sqrt(ρ) 相关的形式，所以空⽓密度降低会增加维持升⼒所需功率。

因此你可以把已有固定⾼度模型改成：


代码块


1 E_h = E_fixed + $\Delta E_{weather}$ + $\Delta E_{density}$ + $\Delta E_{climb}$ 

或者：


代码块


```txt
1 E_h = E_fixed × C_weather(h) + E_climb 
```

其中：


代码块


```txt
1 C_weather(h) = f(headwind_h, crosswind_h, rho_h, T_h, RH_h) 
```

这个 C_weather(h) 可以⽤ XGBoost/LightGBM 学，也可以先⽤物理公式近似。

## 与本项目部署工作的关系

本机无线电地图部署适配目录建议命名为：

```text
modules/radio_map_qcs6490_deploy
```

该目录后续只负责当前 QCS6490/SC171V3 设备上的模型部署、推理、导出、性能测试和结果核验；论文项目源码仍作为模型权重、特征样本和历史结果来源。

## 资料落地规则

- 课件和源码附件先记录原始课程页，不直接批量下载。
- 后续如需下载附件，建议放到对应课程类别目录下的 `attachments/`，再按课程名称建子目录。
- 模型转换、推理和 Fibo AI Stack 相关资料优先放在 `04_AI端侧部署开发/`。
- 与具体示例模型相关的资料按课程类别归档，例如手写数字识别放在 `05_AI端侧部署案例/`。

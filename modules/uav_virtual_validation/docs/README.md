## 当前项目定位

本项目是一个面向城市低空无人机任务的天气驱动能耗预测与虚拟验证平台。

它不是飞控系统，也不是声称完全复现真实世界的高保真数字孪生。当前核心价值是：

- 在禁飞条件下提供可控、可复现的城市低空验证环境。
- 支持真实城市建筑、地面要素、天气采样、航线和禁飞区展示。
- 通过仿真观测与预测输出对比，支撑能耗预测模型的误差评估。
- 为后续接入真实日志、台架实验、PX4/Gazebo 或其他高保真后端保留接口。

## 当前推荐访问方式

开发板或服务器启动后，推荐使用一键脚本输出的地址访问：

```bash
./use_scripts/start_uav_virtual_validation.sh
```

如果浏览器缓存导致页面仍使用旧资源，可以在 URL 后增加或修改 `v=` 参数，也可以使用无痕窗口重新打开。

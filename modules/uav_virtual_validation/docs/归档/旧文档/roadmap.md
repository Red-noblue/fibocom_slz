# UAV 虚拟验证路线图

## 阶段 1：本机简化仿真闭环

目标：先跑通预测、仿真、误差对比、结果输出。

核心产物：

- `sim_summary.json`
- `sim_timeseries.csv`
- `validation_metrics.json`
- `validation_comparison.csv`

## 阶段 2：PX4/Gazebo 接入

目标：引入更真实的飞行动力学、姿态、任务执行和日志输出。

本阶段仍然不做底层飞控开发，只消费仿真状态与日志。

## 阶段 3：高保真三维世界

目标：接入城市建筑、地形、传感器、天气场和三维可视化。

候选平台：

- Gazebo
- AirSim
- Isaac Sim

## 阶段 4：真实数据校准

目标：使用真实飞行日志、台架实验或公开数据对仿真参数做校准。

重点不是宣传“完全真实”，而是量化误差、解释误差来源，并持续缩小 sim-to-real gap。

## 阶段 5：验证体系固化

目标：形成论文、答辩、宣传可复用的证据链。

包括：

- 不同天气场景下的误差表
- 预测曲线与仿真曲线对比
- 风速、温度、载荷、高度消融实验
- 仿真可信度说明

# 验证方法

本项目验证的是任务级天气能耗预测模型，不验证底层飞控控制精度。

## 基本流程

```text
预测输入
  -> 预测输出
  -> 仿真场景配置
  -> 虚拟仿真执行
  -> 仿真真值输出
  -> 预测与仿真误差对比
  -> 展示与报告
```

## 核心指标

- 总能耗误差
- 单位距离能耗误差
- 剩余电量误差
- 风险等级一致性
- 曲线趋势一致性
- 关键事件触发一致性

## 工程原则

- 先保证闭环可运行，再提高仿真精度。
- 所有仿真输出必须可复现。
- 所有误差必须可量化，不使用“极低误差”这类无法证明的表述。
- 后续通过参数辨识和真实日志校准逐步提升可信度。

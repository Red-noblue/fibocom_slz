# 当前能力

## Web 工作台

当前 Web 工作台基于 CesiumJS 构建，默认从本地资产加载城市和 3D Tiles。

主要能力：

- 真实城市切换。
- 本地 3D Tiles 建筑加载。
- 地面要素、航线、天气采样点和禁飞区图层控制。
- 视角遥控器、左键拖动旋转、滚轮缩放。
- 天气采样分辨率控制。
- 建筑、天气点、航线对象点击查看属性。
- 城市要素纠错和版本管理。
- 右键或 `Ctrl/Command + 左键` 多选建筑并批量纠错。
- 太阳投影、AO、基础视觉等显示模式。
- Playwright/Chromium 自动化检查。

## 城市要素纠错

纠错功能用于修正被误识别为建筑的城市对象，不用于做建筑用途分类。

当前支持：

- 单选建筑后修改实际城市要素。
- 右键或 `Ctrl/Command + 左键` 连续多选后批量修改。
- `保存为新版本`、`覆盖当前版本`、`设为默认`、`删除当前版本`。
- 按城市独立显示版本，避免不同城市版本混入。
- 原始版本只读，不能覆盖或删除。
- 非建筑地面要素保存后会生成修正版 3D Tiles，让原错误建筑从正式加载资产中消失。
- `交通立体结构` 和 `交通支撑结构` 保留实体高度，不会被压低为地面薄片。

## 当前边界

- 真实城市建筑来自 OSM，footprint 通常可靠，但高度字段不保证完整。
- 天气数据来自 Open-Meteo，适合城市级和任务级天气背景，不等于建筑尺度 CFD 风场。
- 当前 Web 工作台是验证和展示工具，不是完整飞控地面站。
- 当前仿真链路用于可控验证，不能单独证明真实世界误差极低。

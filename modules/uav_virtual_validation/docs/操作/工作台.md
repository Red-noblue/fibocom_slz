# Web 工作台

## 页面布局

工作台分为三部分：

- 左侧栏：数据源、图层、显示、天气分辨率、视角快捷按钮。
- 中间视图：Cesium 3D 城市场景。
- 右侧栏：世界摘要、视角遥控器、城市要素纠错、选中对象属性。

左右侧栏都可以折叠，用于扩大中间 3D 视图空间。

## 基础交互

- 左键拖动：围绕城市中心旋转和俯仰。
- 滚轮：缩放。
- 左键点击对象：选中建筑、天气点或航线并显示属性。
- 右键点击建筑：加入或移出多选集合。
- `Ctrl/Command + 左键` 点击建筑：加入或移出多选集合。

## 数据源

数据源用于切换城市或程序化世界。

默认真实城市：

```text
真实城市：Manhattan Midtown
```

中国地区建议 Demo：

- 深圳 Futian CBD
- 深圳 Qianhai
- 北京 Zhongguancun
- 广州 Zhujiang New Town
- 上海 Lujiazui
- 苏州 Jinji Lake

## 图层

当前图层包括：

- 城市建筑
- 地面道路
- 禁飞区
- 天气采样
- 计划航线

图层开关影响显示，不会删除底层资产。

## 显示设置

当前默认建议：

- 建筑后端：本地 3D Tiles
- 地形模式：本地平面基底
- 建筑表现：实体+边线
- 光影模式：基础视觉或太阳投影/AO

`Cesium OSM Buildings` 和 `Cesium World Terrain` 需要 Cesium Ion Token。没有 Token 时应优先使用本地 3D Tiles 和本地平面基底。

## 天气分辨率

天气面板区分两类信息：

- 真实源数据：Open-Meteo 历史天气，小时级，水平分辨率约 9-25 km。
- 仿真派生数据：项目在城市范围内派生的三维天气采样点。

当前派生控制：

- 完整 10x6
- 中等 5x3
- 航线走廊 10x2
- 稀疏 4x3

高度层控制：

- 全部 5 层
- 低空 30/60/100m
- 任务 100/150/220m

## 视角遥控器

右侧视角遥控器用于不用鼠标拖动也能控制相机：

- 北/南/东/西：平移目标点。
- 放大/缩小：调整距离。
- 抬头/俯视：调整俯仰角。
- 逆时针/顺时针：绕城市旋转。
- 航线：飞到任务航线。
- 全城：回到城市整体视角。

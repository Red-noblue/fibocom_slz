# 自动化测试

## 测试模块位置

浏览器自动化检查位于：

```text
modules/web_automation_checks/
```

该模块独立于 `uav_virtual_validation`，后续也可以复用给其他 Web 模块。

## 运行命令

```bash
cd modules/web_automation_checks
UAV_WEB_URL='http://127.0.0.1:8090/?v=20260521-move1' npm run check:uav
```

首次运行或 Playwright 版本变化后，需要先安装浏览器运行时：

```bash
cd modules/web_automation_checks
npx playwright install chromium
```

## 检查内容

当前 `check:uav` 会检查：

- 默认模式页面是否加载成功。
- 安全模式是否加载成功。
- 本地 3D Tiles 模式是否加载成功。
- 拖拽视角是否生效。
- 是否出现 `Rendering has stopped`。
- 浏览器控制台是否有错误。
- 天气采样点是否可点击并生成选中效果。
- 天气分辨率切换是否符合预期。
- 太阳投影/AO 模式是否能启用。

## 使用原则

- 修改前端交互、Cesium 渲染、城市资产加载后，至少运行一次 `npm run check:uav`。
- 只做文档修改时不要求运行浏览器检查。
- 如果新增重要交互，应补充定向 Playwright 检查，避免依赖人工反复试错。

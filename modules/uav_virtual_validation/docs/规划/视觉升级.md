# 视觉升级

## 当前视觉方向

当前系统优先保证：

- 城市几何正确。
- 建筑、地面、天气、航线可区分。
- 纠错结果可检查。
- 3D Tiles 加载稳定。
- 避免云彩等会干扰障碍物判断的装饰渲染。

## 当前已实现

- 本地 3D Tiles 建筑。
- 地面 tiles。
- 建筑边线。
- 太阳投影/AO 模式。
- 视角遥控器。
- 纠错对象高亮。
- 多选对象高亮。
- 天气点和航线选中效果。
- 多种纠错显示模式。

## 视觉原则

- 可解释性优先于装饰性。
- 不添加会混淆实体障碍和非实体效果的渲染。
- 建筑边线、阴影和颜色必须服务于空间判断。
- 高对比模式用于检查，柔和模式用于演示。

## 后续升级

建议方向：

- 更高质量地面底图。
- 更精确的地面语义分层。
- 建筑材质和玻璃/混凝土区分。
- 城市级标签和重点对象标注。
- 任务结果播放时间轴。
- 能耗曲线与 3D 轨迹联动。
- 大城市分块加载和 LOD 优化。

## 不建议方向

短期不建议：

- 加回云层和雾效。
- 加入会遮挡建筑边界的强景深或模糊。
- 为了美观牺牲障碍物可识别性。
- 直接依赖在线 Token 服务作为默认展示路径。

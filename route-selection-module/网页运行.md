4. 点击“生成候选航线”。

## 4. 仅建筑模式规则

`building_only` 当前规则：

- 最大高度是硬约束，填写 `27m` 就不会生成超过 `27m` 的航线。
- 最大高度填得很低时，会优先走低空楼间/道路廊道穿梭。
- 路径不能穿建筑本体。
- 在高度约束范围内允许升降。
- 不再因为“达不到建筑净空建议高度”直接失败。
- 如果建筑本体几何上完全堵死，才会返回不可行。

输出固定 5 类路线：

- 最快到达
- 低风险
- 能耗最少
- 均衡稳定推荐
- 最畅通路线

## 5. 综合模式规则

`combined` 当前规则：

- 综合页面：`http://10.129.198.166:8091/web/route_selection_combined.html`
- 综合候选航线同时考虑天气风险和建筑避障。
- 页面会叠加展示建筑模型、避障航线管体、降雨/风险热力区、风向、云量、湍流等天气态势。
- 候选航线展示的首航点和末航点固定等于输入框里的起点/终点。
- 如果输入起点刚好落在建筑 footprint 内，会保留它作为起飞锚点；建筑硬避障从离场后的可飞走廊开始执行。
- 中间航线段仍按建筑 footprint 硬避障，不穿建筑本体。
- 低风险路线会优先绕开红色高风险天气区；如果确实无法完全绕开，才允许少量穿越。
- 其他路线会按策略取舍：最快到达、能耗最少、均衡稳定、最畅通路线可能会保留更短距离或更好通达性，因此天气暴露可能高于低风险路线。

输出同样固定 5 类路线：

- 最快到达
- 低风险
- 能耗最少
- 均衡稳定推荐
- 最畅通路线

## 6. 快速自检

页面打不开时先测页面：

```bash
python3 - <<'PY'
import urllib.request
for path in ["/web/route_selection_building.html", "/route-selection-plugin.js", "/api/route-selection/datasets"]:
    r = urllib.request.urlopen("http://10.129.198.166:8091" + path, timeout=10)
    print(path, r.status, r.headers.get("Content-Type"))
PY
```

低高度航线自检，验证上海、深圳、曼哈顿都能在 `max_altitude_m=27` 下生成 5 条：

```bash
python3 - <<'PY'
import json, time, urllib.request
base = "http://10.129.198.166:8091"
datasets = json.load(urllib.request.urlopen(base + "/api/route-selection/datasets"))["datasets"]
by_key = {item["dataset_key"]: item for item in datasets}
for key in ["shanghai_people_square", "shenzhen", "manhattan"]:
    item = by_key[key]
    start = item["default_route"][0]
    end = item["default_route"][-1]
    payload = {
        "dataset_key": key,
        "planning_mode": "building_only",
        "start_lat": start["lat"],
        "start_lon": start["lon"],
        "end_lat": end["lat"],
        "end_lon": end["lon"],
        "start_altitude_m": 15,
        "end_altitude_m": 10,
        "min_altitude_m": 0,
        "candidate_count": 5,
        "cell_m": 220,
        "safety_clearance_m": 25,
        "max_altitude_m": 27,
    }
    req = urllib.request.Request(
        base + "/api/route-selection/plan",
        data=json.dumps(payload).encode(),
        headers={"Content-Type": "application/json"},
        method="POST",
    )
    t = time.time()
    data = json.load(urllib.request.urlopen(req, timeout=90))
    routes = data.get("routes", [])
    max_alt = max(float(p["altitude_m"]) for r in routes for p in r["waypoints"])
    print(key, "routes=", len(routes), "max_alt=", max_alt, "elapsed=", round(time.time() - t, 2))
PY
```

正常输出应类似：

```text
shanghai_people_square routes= 5 max_alt= 27.0
shenzhen routes= 5 max_alt= 27.0
manhattan routes= 5 max_alt= 27.0
```

## 7. 常见问题

如果页面还是旧效果：

- 确认你访问的是 `8091`，不是旧独立页端口。
- 重启 `serve_uav_workbench_route_selection.py`。
- 浏览器按 `Ctrl + Shift + R` 强制刷新。
- 确认 `/route-selection-plugin.js` 返回的是最新文件。

如果点“生成候选航线”失败：

- 先跑第 6 节接口自检。
- 如果接口能生成，问题在浏览器缓存或前端状态。
- 如果接口失败，把接口返回的 `error` 贴出来继续查。
